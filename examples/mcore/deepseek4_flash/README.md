# DeepSeek-V4模型训练

重磅预告！基于昇腾硬件的大语言模型套件MindSpeed LLM，即将正式上线DeepSeek-V4预训练支持功能✨ 此次适配将充分融合两者核心优势，为开发者带来更高效、更强大的模型训练体验。 

MindSpeed LLM作为端到端的大语言模型套件，具备完善的分布式训练能力与丰富的加速特性，可高效支撑各类大模型的预训练与微调任务；而DeepSeek-V4作为全新开源模型，拥有百万Token超长上下文、领先的Agent能力与推理性能，在技术架构上实现多项关键突破。两者的强强联合，将进一步降低超大规模模型预训练的门槛，解锁更多高阶训练场景。 

目前，MindSpeed LLM支持DeepSeek-V4预训练功能正处于最终筹备阶段，各项性能调试已接近尾声。更多适配细节与使用指南，将在上线后同步公布，敬请各位开发者与行业伙伴期待！
